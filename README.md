# Projektuppgift

## Beskrivning
CineSearch är en webbapplikation där användaren kan söka efter filmer och få detaljerad information som betyg, beskrivning och språk.
Data hämtas från två olika API:er – OMDb och TMDB. 
Användaren kan även spara favoritfilmer som lagras mellan sessioner med localStorage. 
Webbplatsen är byggd med HTML, SCSS och JavaScript utan frontend-ramverk, och använder Vite som automatiserad arbetsprocess.
